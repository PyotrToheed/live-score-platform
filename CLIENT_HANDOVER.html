4. Admin User Guide

How to Sync New Matches

  1. Login to /admin/dashboard.
  2. Locate the Premier League card.
  3. Click "Sync Fixtures & Odds".
  4. Wait ~10 seconds for completion.
  5. Check the live site to see new matches appear in all languages.
